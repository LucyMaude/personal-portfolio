runability is a full-stack app i made to help aspiring runners like myself train for their upcoming race(s). I used Mongodb and Express to give the user the ability to log-in and create training schedules for their upcoming races.

runability has been deployed! you can click here to take a look;

rock-the-vote:

I created this fun static app as a project for class. I used a lot of CSS stylings to give rock-the-vote an extra patriotic theme. Users can submit an issue to be voted/commented on by their peers and issues will display in order by least number of votes. To make sure these issues persisted a page-refresh, I used express and nodejs.

apiofthrones:

apiofthrones is an app that displays information about the book/tv series Game of Thrones. It was created to demonstrate my understanding of a.p.is, and have some fun with styling. the user can see information about the different "houses" and characters within the world created by George R. R. Martin, and even includes a fun quiz that i made!
